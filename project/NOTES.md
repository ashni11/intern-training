# pip vs uv

## pip
- Traditional Python package manager
- Uses: pip install package_name
- Widely used

## uv
- Faster than pip
- Creates virtual environments quickly
- Uses: uv pip install package_name

## Comparison

| Feature | pip | uv |
|----------|-----|-----|
| Speed | Slower | Faster |
| Popularity | Common | New |
| Environment Creation | python -m venv | uv venv |

Conclusion:
uv is faster, while pip is more widely used.
